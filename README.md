# 🚗 CarFix: دستیار هوشمند عیب‌یابی خودرو (RAG)

**توضیح کلی:**
این پروژه یک سیستم پرسش و پاسخ هوشمند (RAG) است که برای عیب‌یابی خودروهای مدرن طراحی شده است. کاربر با وارد کردن کد خطای OBD-II یا شرح علائم خودرو، پاسخ دقیق (علت خطا و راه‌حل تعمیر) را دریافت می‌کند.

**معماری و نوآوری (نسبت به RAG معمولی):**
1. **فیلتر هوشمند متاداده (Metadata Filtering):** تشخیص برند و مدل خودرو در لحظه و محدود کردن جستجوی پایگاه داده فقط به اسناد مربوط به آن خودرو.
2. **جستجوی هیبریدی (Hybrid Search):** ترکیب جستجوی کلمه‌ای (برای پیدا کردن کدهای دقیق مثل P0171) و جستجوی وکتوری (برای درک معنای متن).
3. **لایه اعتبارسنجی:** مدل موظف است پاسخ خود را تنها بر اساس اسناد موجود بدهد و در صورت نبود اطلاعات، اعلام کند "اطلاعات کافی نیست" (جلوگیری از توهم).

**تکنولوژی‌های استفاده شده:**
- LangChain (برای مدیریت پایپلاین RAG)
- FAISS (برای پایگاه داده وکتوری)
- Sentence-Transformers (برای تولید Embedding)
- Streamlit (برای رابط کاربری وب)
- PyPDF2 / Unstructured (برای خواندن PDF)

**وضعیت فعلی:**
تاییدیه استاد گرفته شده است. در حال آماده‌سازی دیتاست و پیاده‌سازی هستیم.
